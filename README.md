# RailsBlog-rubyschool
Rails Blog (rubyschool project). Ruby on Rails.

Учебный проект на Ruby/Sinatra для разбора как работает Ruby on Rails.

## Start project

1. Install gems:

```bash
bundle install
```

2. Migrate:

```bash
rake db:migrate
```

3. Run RoR application:

```bash
rails s
```

4. Open in your browser: http://localhost:3000/
